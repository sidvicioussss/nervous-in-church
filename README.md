# Nervous in Church

A quiet room for legal writing.

## Philosophy

Not Word. Not Notion. No UI noise. Just a place to write.

## v0.1 Features

- Full-screen editor with Claude-style dark theme
- Plain text editing (no live markdown rendering)
- Keyboard-driven file operations:
  - `Cmd+N` — New file
  - `Cmd+O` — Open .md file
  - `Cmd+S` — Save with YAML frontmatter
  - `Cmd+Shift+S` — Save As
  - `Cmd+E` — Export to .docx or .pdf

## Requirements

- Node.js 18+
- pandoc (for export functionality)

## Install

```bash
npm install
```

## Run

```bash
npm start
```

## Build

```bash
npm run build
```
